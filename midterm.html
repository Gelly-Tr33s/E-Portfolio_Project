    Activity 5 is an individual activity. We were tasked to design and develop static webpage for an event or conference, using HTML and CSS. The content of the webpage is all fictional and made-up. It contains the following:

  • Header (or the navbar), Hero section to captivate audiences
  • About section which contains information about the event
  • Speakers section that has the profiles of speakers and brief description of their backgrounds
  • Venue section that shows the address and a few pictures
